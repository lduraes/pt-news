## Key features
* Vanilla **SwiftUI** + **SPM** + **MVVM** implementation
* Decoupled **Presentation** and **Business Logic** layers
* Simple flexible networking layer built on Generics
* Development mindset related to SOLID, DRY & KISS concepts
